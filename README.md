## General info
This project is an automation of product selection and top buttons validation of website www.revzila.com.
	
## Technologies
Project is created with:
* Java: 15.0.1
* Eclipse IDE: 4.17.0
	
## Setup
To run this project, download it and run it locally using cmd:

```
$ cd ../RevZilla
$ mvn clean
$ mvn compile
$ mvn test
```
